### 一、基本语法
1. **标题**
   #一级标题
   ##二级标题
2. **引用**
   > 这是一个引用
3. **列表**   
   1. 无序标题
   - 列表1
   + 列表2
   * 列表3
   2. 有序列表
   3. 嵌套
   4. Todo List
      - [x] Become a millionaire.
      - [x] Get a girlfriend.
      - [ ] Never lie.
4. **表格**
   | 左对齐 | 居中对齐 | 右对齐 |
   | :---- | :-----: | ----: |
   | a | b | c |
5. **段落**
   - 换行：两个及以上空格后回车/空一行  

     abc
   - 分割线：三个***
     ***
   - 字体
     | 字体 | 代码 |
     | :--: | :--: |
     | *斜体* | * * |
     | ==高亮== | == == |
     | **粗体** | ** ** |
     | ***粗斜体*** | *** *** |
     | ~~删除~~ | ~~ ~~ |
     | <u>下划线</u> |``` <u> </u> ```|
   - 脚注
     这是一个脚[^1]注。
